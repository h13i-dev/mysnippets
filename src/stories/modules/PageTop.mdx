# ページトップボタン

## 概要
ページ上部へスクロールするためのボタンを実装するモジュールです。スクロール位置に応じた表示制御と、フッター領域での位置調整機能を提供します。

[デモページ](demo/pagetop/)

### HTMLコード
```html
<a class="pagetop" href="#pagetop">ページトップへ</a>
```

### 使い方
```javascript
import PageTop from "@assets/js/modules/PageTop";

new PageTop(".pagetop", {
  threshold: 200,
  activeClass: 'is-active',
  bottomClass: 'is-bottom',
  target: 'footer',
  fixed: true
});
```

### 主な機能
- スクロール位置に応じたボタンの表示/非表示
- フッター領域での位置調整（フッターに重ならないように）
- カスタマイズ可能な表示閾値
- aria-hidden属性による適切なアクセシビリティ対応
- スクロールとリサイズイベントの自動監視

## オプション

### target

フッター等の基準要素のセレクターを指定します。この要素に近づくとボタンがabsoluteポジションに切り替わります。
```js
new PageTop(".pagetop", {
  target: "footer",
});
```

### threshold

スクロール距離がこのピクセル値に達するとボタンが表示されます。デフォルトは200pxです。
```js
new PageTop(".pagetop", {
  threshold: 500,
});
```

### activeClass

ボタン表示時に付与されるクラス名を指定できます。デフォルトは'is-active'です。
```js
new PageTop(".pagetop", {
  activeClass: "is-visible",
});
```

### bottomClass

基準要素（フッターなど）到達時に付与されるクラス名を指定できます。デフォルトは'is-bottom'です。
```js
new PageTop(".pagetop", {
  target: "footer",
  bottomClass: "is-at-bottom",
});
```

### fixed

基準要素の手前で固定するかどうかを指定します。デフォルトはtrueです。falseにすると常に画面右下に固定表示されます。
```js
new PageTop(".pagetop", {
  target: "footer",
  fixed: false,
});
```
