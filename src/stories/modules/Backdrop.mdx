# オーバーレイ

## 概要
クリックでオーバーレイを表示・非表示できるモジュールです。クリック、キーボード操作、スクロール検知などを適切に処理し、アクセシビリティ属性を維持します。

[デモページ](demo/backdrop/)

### HTMLコード
```html
<div class="c-backdrop js-backdrop" aria-hidden="true"></div>
<button type="button" class="js-backdrop-trigger">オーバーレイを開く</button>
```

### 使い方
```javascript
import Backdrop from "@assets/js/modules/Backdrop";

new Backdrop({
  button: ".js-backdrop-trigger",
  backdrop: ".js-backdrop",
  closeButton: ".js-close-btn",
  closeOnResize: false,
  excludeClose: [".header"],
  activeClass: "is-active"
});
```

### 主な機能
- ボタンクリックでオーバーレイの開閉
- ESCキーで閉じる
- オーバーレイ外クリックで閉じる
- 除外エリア設定（ヘッダーなど特定のエリアをクリックしても閉じない）
- aria-expanded属性による適切なアクセシビリティ対応
- リサイズ時の自動クローズオプション

## オプション

### button

トグルボタンのセレクターを指定します（必須）。
```js
new Backdrop({
  button: ".js-backdrop-trigger",
  backdrop: ".js-backdrop",
});
```

### backdrop

オーバーレイのセレクターを指定します（必須）。
```js
new Backdrop({
  button: ".js-backdrop-trigger",
  backdrop: ".js-backdrop",
});
```

### closeButton

閉じるボタンのセレクターを指定します。オーバーレイ内にナビゲーションなどが含まれる場合に便利です。
```js
new Backdrop({
  button: ".js-backdrop-trigger",
  backdrop: ".js-backdrop",
  closeButton: ".js-close-btn",
});
```

### excludeClose

クリックイベントを無視するエリアのセレクターを配列で指定します。ヘッダーなど特定のエリアをクリックしてもオーバーレイが閉じないようにできます。
```js
new Backdrop({
  button: ".js-backdrop-trigger",
  backdrop: ".js-backdrop",
  excludeClose: [".header", ".sidebar"],
});
```

### closeOnResize

windowリサイズ時にオーバーレイを閉じるかどうかを設定します。デフォルトはfalseです。
```js
new Backdrop({
  button: ".js-backdrop-trigger",
  backdrop: ".js-backdrop",
  closeOnResize: true,
});
```

### activeClass

アクティブ時に付与されるクラス名を指定します。デフォルトは'is-active'です。
```js
new Backdrop({
  button: ".js-backdrop-trigger",
  backdrop: ".js-backdrop",
  activeClass: "is-open",
});
```
