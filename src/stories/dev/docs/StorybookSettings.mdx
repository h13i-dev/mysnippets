# Storybookの設定

プロジェクト固有のStorybook設定について説明します。

## HTML変換機能（useHtmlTransform）

ストーリーのソースコード表示をJSXからHTMLに変換する機能です。

### グローバル設定

`src/stories/assets/utils/htmlTransform.ts`で全体の動作を制御できます。

```typescript
export const useHtmlTransform = false; // 全体で無効化（デフォルト）
```

### 個別ストーリーでの設定

#### 方法1: `createHtmlSource`の第2引数で指定

```typescript
import { createHtmlSource } from "@stories/assets/utils/htmlTransform";

const meta: Meta<typeof Component> = {
  parameters: {
    docs: {
      source: createHtmlSource("dynamic", true), // グローバル設定を上書きして有効化
    },
  },
};

export const Story: Story = {
  parameters: {
    docs: {
      source: createHtmlSource("dynamic", false), // このストーリーだけ無効化
    },
  },
};
```

#### 方法2: `parameters.useHtmlTransform`で指定

```typescript
export const Story: Story = {
  parameters: {
    useHtmlTransform: false, // このストーリーだけ無効化
    docs: {
      source: createHtmlSource("dynamic"),
    },
  },
};
```

### `createHtmlSource`の引数

#### 第1引数: `sourceType`

- `"dynamic"`: argsでコード表示する場合、またはComponent.Name以外のコンポーネントをrenderで表示する場合
- `"code"`: Component.Nameのコンポーネントをrenderでコード表示する場合（コンポーネント部分のみ抽出）

#### 第2引数: `enableHtmlTransform`（省略可）

- `true`: HTML変換を有効化
- `false`: HTML変換を無効化
- 省略時: グローバル設定（`useHtmlTransform`）を使用

### 優先順位

設定の優先順位は以下の通りです（上が優先）：

- **1位**: `parameters.useHtmlTransform`
- **2位**: `createHtmlSource`の第2引数
- **3位**: グローバル設定（`useHtmlTransform`）

### 使用例

```typescript
import type { Meta, StoryObj } from "@storybook/react";
import { Heading } from "@components/index.tsx";
import { createHtmlSource } from "@stories/assets/utils/htmlTransform";

const meta: Meta<typeof Heading> = {
  title: "Components/見出し",
  component: Heading,
  tags: ["autodocs"],
  parameters: {
    docs: {
      source: createHtmlSource("dynamic"), // グローバル設定を使用
    },
  },
};

export default meta;
type Story = StoryObj<typeof Heading>;

// このストーリーだけHTML変換を有効化
export const HeadingLv1: Story = {
  name: "見出しレベル1",
  parameters: {
    docs: {
      source: createHtmlSource("dynamic", true),
    },
  },
  args: {
    lv: "1",
    children: "見出しレベル1",
  },
};

// グローバル設定に従う（無効）
export const HeadingLv2: Story = {
  name: "見出しレベル2",
  args: {
    lv: "2",
    children: "見出しレベル2",
  },
};
```
