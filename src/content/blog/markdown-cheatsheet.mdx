## リスト（Lists）

### 番号なしリスト

<Code lang="markdown">{`
- リストアイテム1
- リストアイテム2
  - ネストしたアイテム
  - ネストしたアイテム2
    - ネストしたアイテム
    - ネストしたアイテム2
- リストアイテム3

* 別の記法
`}</Code>

- リストアイテム1
- リストアイテム2
  - ネストしたアイテム
  - ネストしたアイテム2
    - ネストしたアイテム
    - ネストしたアイテム2
- リストアイテム3

* 別の記法

### 番号付きリスト

<Code lang="markdown">{`
1. 最初のアイテム
2. 2番目のアイテム
  1. ネストした番号付きリスト
  2. ネストした番号付きリスト2
    1. ネストした番号付きリスト
    2. ネストした番号付きリスト2
3. 3番目のアイテム
`}</Code>

1. 最初のアイテム
2. 2番目のアイテム
   1. ネストした番号付きリスト
   2. ネストした番号付きリスト2
      1. ネストした番号付きリスト
      2. ネストした番号付きリスト2
3. 3番目のアイテム

## コードブロック（Code Blocks）

### 基本的なコードブロック

### 言語指定あり

<Code lang="astro">{`

\`}</Code>

\`}</Code>
`}</Code>

<Code lang="diff">{`

`}</Code>

## 表（Tables）

<Code lang="markdown">{`
| Head | Head | Head |
| ---- | ---- | ---- |
| Text | Text | Text |
| Text | Text | Text |

| 左揃え | 中央揃え |  右揃え |
| :----  | :-----: | -----: |
| 左     |   中央   |     右 |
`}</Code>

| ヘッダー1 | ヘッダー2 | ヘッダー3 |
| --------- | --------- | --------- |
| セル1     | セル2     | セル3（長文テキストが入ります）     |
| セル4     | セル5     | セル6     |

| 左揃え | 中央揃え | 右揃え |
| :----- | :------: | -----: |
| 左     |   中央   |     右 |
