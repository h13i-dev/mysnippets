# AGENTS.md

## テスト

- ユーティリティ、hooks、独立したモジュールのユニットテストには `*.spec.ts` / `*.spec.tsx` を使う。
- 複数モジュールを組み合わせるインテグレーションテストには `*.test.ts` / `*.test.tsx` を使う。
- 正常系、異常系、境界値を網羅する。
- E2E テストは開発サーバーを起動した状態で実行する。
- コミット前相当の検証では `npm run axe` と `npm run nu` を実行する。
- `package.json` に VRT script が追加された場合、コンポーネントのスタイル変更時に VRT を実行し、意図した変更の場合だけ snapshot を更新する。
